## 🎓 新手入门指南

### 如何添加新节点

1. 创建节点文件：`dataengineer/agent/node/my_node.py`

```python
from datus.agent.node.node import Node


class MyNode(Node):
    async def execute(self, state: dict) -> dict:
        # 实现节点逻辑
        return state
```

2. 注册节点类型：`dataengineer/configuration/node_type.py`

```python
class NodeType:
    MY_NODE = "my_node"
```

3. 添加到工厂：`dataengineer/agent/node/node.py:new_instance()`

```python
from datus.agent.node.my_node import MyNode

# 在工厂方法中添加映射
```

### 如何添加新存储模块

1. 继承基类：`dataengineer/storage/base.py:BaseEmbeddingStore`
2. 定义 Schema：使用 PyArrow 定义表结构
3. 实现 CRUD：`store_batch`、`search`、`delete`
