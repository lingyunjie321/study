🎓 新手入⻔指南

如何添加新节点
1. 创建节点文件: datus/agent/node/my_node.py

```text
  1 from datus.agent.node.node import Node
```

2

```text
  3 class MyNode(Node):
  4  async def execute(self, state: dict) -> dict:
  5    # 实现节点逻辑return state
```

2. 注册节点类型: datus/configuration/node_type.py

```text
  1 class NodeType:
  2  MY_NODE = "my_node"
```

3. 添加到工厂: datus/agent/node/node.py:new_instance()

```text
  1 from datus.agent.node.my_node import MyNode
  2 # 在工厂方法中添加映射
```

如何添加新存储模块
1. 继承基类: datus/storage/base.py:BaseEmbeddingStore
2. 定义 Schema:使用 PyArrow 定义表结构

3. 实现 CRUD:store_batch、search、delete
