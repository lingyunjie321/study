## 新手入门指南

### 如何添加一个新的文档格式支持

1. **注册扩展名**：在`agents/doc_parser_agent.py`的`SUPPORTED_EXTENSIONS`字典中添加映射
2. **实现解析方法**：添加`_parse_xxx(self, file_path) -> list[str]`方法
3. **添加路由分支**：在`parse()`方法的if-elif链中添加对应分支

示例——添加`.docx`支持：
```python
# 1. SUPPORTED_EXTENSIONS 添加
".docx": DocType.TEXT,

# 2. 添加解析方法
async def _parse_docx(self, file_path: str) -> list[str]:
    import docx
    doc = docx.Document(file_path)
    return [p.text for p in doc.paragraphs if p.text.strip()]

# 3. parse() 中添加分支
elif doc_type == DocType.TEXT and file_path.endswith(".docx"):
    raw_texts = await self._parse_docx(file_path)
```

### 如何调整检索策略

在`agents/qa_agent.py:233`的`_hybrid_rerank`方法中调整权重：
```python
weight_map = {"vector": 1.0, "graph": 1.2, "hybrid": 1.1}
# 提高图谱权重 → 更重视结构化关系
# 降低图谱权重 → 更重视语义相似度
```
